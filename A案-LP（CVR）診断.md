# ハートフルクリーン LP — 辛口レビュー（Design Critic × LP-OS）

## コンテキスト

| 項目 | 内容 |
|------|------|
| **ターゲット** | 遺品整理を初めて依頼する家族（40〜60代、不安・時間的制約あり） |
| **目的** | 電話/LINE問い合わせ（CVR最大化） |
| **業種・価格帯** | 遺品整理・特殊清掃（BtoC・中単価） |
| **ベンチマーク** | 業界トップのLPO・信頼構造型LP |

## 3秒テスト

| 項目 | 評価 |
|------|------|
| **スコア** | 5/10 |
| **第一印象** | 「緑が多くて企業サイト感はあるが、ヘッダーにロゴが無い・FVの写真の切り抜きが不自然で、一瞬で信頼を損なっている」 |
| **離脱衝動** | ロゴ欠如と不自然なビジュアルで「ちゃんとした会社か？」と疑い、閉じる可能性が高い |

## 致命的な問題

🔴 **1. ヘッダーにロゴ・社名が表示されない（または極端に目立たない）**  
→ 3秒で「何のサイトか」が伝わらず、離脱とブランド記憶の喪失を招く。

🔴 **2. FVの代表写真が object-fit: cover + object-position: center で顔が切れている**  
→ 人物の信頼性を伝えるはずの写真が「不自然な切り抜き」になり、逆効果。

🔴 **3. メインCTA（オレンジボタン）の可読性不足**  
→ 「今すぐ電話で相談」が読みにくいと、第一印象でアクションが取られない。

---

## 改善指示（優先順位順）

🔴 1. **ヘッダー**  
- ロゴ画像の `src` を絶対パスまたは確実な相対パスに。  
- ロゴが読み込めない場合のフォールバックとして、社名テキストを必ず表示（`font-size: 16px` 以上、`font-weight: 600`）。  
- ロゴ＋社名を一塊でクリック可能にし、タップ領域 44px 以上を確保。

🔴 2. **FV画像**  
- `.fv__bg img` に `object-position: 28% 50%;` または `object-position: left center;` を指定し、縦長ポートレートで「顔」が必ず入るようにする。  
- 必要なら `object-fit: cover` のまま、ブレークポイントごとに position を微調整。

🔴 3. **メインCTA**  
- `.btn--primary` の最小高さを `48px` に。  
- 文字に `text-shadow: 0 1px 2px rgba(0,0,0,0.15);` と `font-weight: 600` を付け、オレンジ地でも可読性を確保。

🟡 4. **セクション余白**  
- LP-OS「視覚的リズム」に従い、セクション間は `padding: clamp(100px, 12vw, 160px) 0` 程度に統一し、120px〜200px の範囲を死守。

🟡 5. **CTAの一貫性**  
- メインは「電話」か「LINE」のどちらか1本に絞り、もう一方はセカンダリに。  
- 全ページで「無料」「〇〇するだけ」などのマイクロコピーを統一。

🟢 6. **権威バッジの位置**  
- FV内で「Google口コミ★5.0」を、キャッチの直下またはCTAの上に固定し、3秒で目に入るようにする（現状で近い場合は微調整のみ）。

---

## 唯一の良い点

キャッチコピー「追加請求ゼロ。見積り後の値段で、その日じゅうに。」は、不安を言語化しており、LP-OSの「ベネフィット＋具体性」を満たしている。
