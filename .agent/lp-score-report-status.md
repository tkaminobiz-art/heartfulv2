## 今回の実装で対応した項目（まとめ）

- **⑦ 受付時間**：7:00〜22:00に統一（既に反映済みのため再確認のみ）
- **⑧ 構造化データ**：LocalBusiness・FAQPage の JSON-LD を追加
- **⑧ OGP**：og:* と Twitter Card、canonical を追加（本番URL・og:image は要設定）
- **⑧ 遅延読み込み**：FV以外の画像に `loading="lazy"` を付与
- **⑨ canonical・OGP**：タグ追加（値は要設定）

---

## 本番公開前に要設定する値

1. **og:url** ・ **og:image** ・ **canonical**：公開するLPの絶対URLと、SNS用画像の絶対URL
2. **JSON-LD の "url"**：LocalBusiness の公式サイト／LPのURL（必要なら）
3. **画像が表示されない場合**：フォルダ「ハートフルクリーンサイト画像」を「images」等のASCII名にリネームし、`index_b.html` 内のパスを一括置換
