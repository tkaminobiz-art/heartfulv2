# index_b.html デザイン採点レポート

## コンテキスト定義

| 項目 | 内容 |
|------|------|
| **ターゲット** | 40〜50代、親の遺品整理で悩むご遺族。悲しみ・不安・「業者を選ぶ自信がない」という心理状態。 |
| **目的** | 問い合わせ・電話・LINEへの誘導（CVR）。信頼獲得と「この会社に頼みたい」という決断。 |
| **業種・価格帯** | 遺品整理・生前整理。高単価・人生の一大事なので「安さ」より「安心」が軸。 |
| **ベンチマーク** | 同業の信頼感のあるLP、Awwwards の Corporate / Landing で「落ち着いた信頼感」を出しているサイト。 |

## 違和感の言語化（なぜ「とてもクライアントに見せられない」と感じるか）

### 1. 「部品の寄せ集め」感
- セクションごとに**レイアウトの型がバラバラ**（中央揃え・グリッド・フレックス・インラインスタイル多用）で、**一つの世界観として成立していない**。
- 見出しの下の「オレンジの短い線」、カードの角丸・ボーダー、CTAのグラデーションなどが**同じパターンで繰り返される**が、**余白・フォントサイズ・色のトーン**がセクション間で揺れており、「ひとつのLP」ではなく「複数のテンプレを貼り合わせた」印象になる。

### 2. メリハリの不在
- **情報の優先度が視覚的に伝わらない**。すべての見出しが同じような大きさ・ウェイトで、**どこを読めばいいかが3秒で分からない**。
- ネイビーの左右と白い中央という**枠組みはある**が、**中身のコントラスト**（大きい／小さい、太い／細い、色の強弱）が弱く、**平坦で飽きる**。

### 3. 「信頼」の見せ方が古い
- **バッジ・タグ・約束の羅列**が多く、**感情に訴えるビジュアルやストーリー**が少ない。2010年代の「箇条書きLP」のまま進化していない。
- 代表写真・Before/After・お客様の声はあるが、**配置とサイズが控えめ**で、**「この人に任せたい」という感情が湧く構図**になっていない。

### 4. 密度と余白の不統一
- **padding: 40px 20px** が全セクションに共通で、**セクションの重要度や内容量に応じた呼吸**がない。重要なKVやCTAも、地味な「よくある質問」も同じリズムで並ぶ。
- **max-width: 750px** と **padding: 0 20px** が混在し、**実際の「読ませる幅」や「タップしやすさ」**がセクションごとに揺れている。

### 5. 色とトーンの雑多さ
- オレンジ・ネイビー・ゴールド・赤・緑・グレー・クリームが**同時に**使われ、**主役の色**が定まらない。**「このサイトの色はこれ」**が一言で言えない。
- ネイビーは「左右のフレーム」として使っているが、**中央のセクション背景**（白・クリーム・薄ピンク・薄黄）が**バラバラに切り替わり**、**色の物語**がなく「とりあえず変えた」感がある。

### 6. インタラクションの欠如
- **ホバーは opacity: .9 程度**のみ。ボタン・カード・リンクに**「触れた・押した」というフィードバック**がほとんどなく、**画面が静的な紙**のようで、**Webらしさがない**。

## モダンでない作りの原因の言語化

### A. レイアウトが「テンプレ的」
- **中央寄せ＋均等グリッド＋同じパディング**の繰り返し。**非対称・Bento・余白の大胆さ・画像とテキストのスケール差**など、**2020年代以降のLPで当たり前の文法**が入っていない。
- **750px 固定**は「読みやすい」が、**大画面でどう見せるか**（余白を活かす・写真を大きく見せる）の**意図が感じられない**。

### B. タイポグラフィに階層がない
- Noto Serif / Noto Sans は選んでいるが、**見出しと本文のサイズ差・ウェイト差が弱い**。**2.6rem の h2 が多く**、**「ここが一番大事」という見出し**が際立っていない。
- **行間・字間**は指定されているが、**長文ブロック**での**読みやすさの設計**（段落間・引用の扱い）が弱い。

### C. コンポーネントの「質」が揃っていない
- カードは **border + padding + border-radius** で作っているが、**影・ホバー・フォーカス**がなく**平面的**。**1枚のカードにどれだけ情報を載せるか**のルールもなく、**詰め込み**と**スカスカ**が混在している。
- CTAボタンが**緑のグラデーション＋黄色バッジ**で**派手**な一方、フォーム送信ボタンは**オレンジ単色**で、**同じ「行動」の重要度がビジュアルで統一されていない**。

### D. 画像の使い方が「挿し絵」止まり
- 画像は**セクションの上や横に置いてある**だけで、**写真がストーリーの主役**になる構図（大きく1枚・テキストと重ねる・流れの中で見せる）になっていない。
- **フルスクリーン表示**は指定しているが、**高さやアスペクト比の統一**がなく、**セクションごとにバラつき**やすい。

### E. アニメーション・マイクロインタラクションがほぼない
- **スクロールで浮くフロートバー**と**FAQの開閉**以外、**動きがない**。**出現・スタガー・ホバー・スクロール連動**など、**「見ているだけで少し気持ちいい」体験**がなく、**静的で古い**印象になる。

### F. インラインスタイルとクラスが混在
- **style="..."** が多数あり、**デザインの変更や一括調整**がしづらい。**「この色はここでしか使っていない」**が多く、**デザインシステム**になっていない。

## 致命的な問題（最優先で直すべき3つ）

1. **情報の優先度が視覚的に伝わっていない**  
   FV・見出し・CTA の「どこを一番見てほしいか」が、サイズ・余白・色で明確でない。**h1 と CTA を際立たせ、他は一段下げる**階層設計が必要。

2. **「一つのLP」としての世界観がない**  
   セクションごとに背景色・パディング・カードの形が揺れ、**統一されたトークン（余白の倍数・角丸・影・色の数）**がない。**デザインシステムを決め、インラインをやめてクラスで統一**すべき。

3. **信頼と感情を伝えるビジュアルが弱い**  
   代表・Before/After・お客様の声が**小さく・同じリズムで並んでいる**。**「この人に任せたい」と思わせる**ために、**代表とメッセージを大きく見せる**・**Before/After を1セクションの主役にする**などの**構図の見直し**が必要。

---

## 改善指示（優先順位順）

1. **[致命的] 視覚階層の設計**  
   - FV: h1 を大きく（例: clamp で 4rem 前後）、サブとバッジを一段下げる。CTA を 1 つに絞り、サイズと色で最優先にする。  
   - 全セクション: h2 を 2.6rem に揃えず、「メインの見出し」だけ 3rem 以上・太めにし、他は 2rem 前後で従にする。

2. **[致命的] カラーパレットの整理**  
   - メインは **ネイビー・オレンジ・白（＋クリーム1種）** に絞る。赤・ゴールド・緑は **CTA・アクセント** に限定し、背景や枠には使わない。  
   - セクション背景は **白 / クリームの2トーン** に統一し、ネイビーは左右フレームと「なぜ元警察官」のみにする。

3. **[致命的] 代表・Before/After・お客様の声の「主役化」**  
   - 代表: 写真を大きく（例: 200px 以上）、コピーと並べるレイアウトを維持しつつ、余白を増やして「この人」が目に入るようにする。  
   - Before/After: 1〜2ケースを大きく見せ、残りはサムネイルやスライダーで補助にする。  
   - お客様の声: 1件を大きく引用として見せ、Google 評価を併記するブロックを 1 つ作る。

4. **[重要] 余白のルール化**  
   - セクション間: 例として 64px / 96px / 120px の3段階を決め、重要度で使い分ける。  
   - セクション内: padding を 24px / 32px / 48px など **8px の倍数** に統一し、max-width と padding の組み合わせを 1 パターンに寄せる。

5. **[重要] カード・CTA のコンポーネント化**  
   - カード: クラスで border-radius・padding・影・ホバー（軽い lift や shadow）を統一。  
   - CTA: プライマリ（電話/LINE）とセカンダリ（フォーム）で色・サイズを分け、全ページで同じ見せ方にする。

6. **[重要] インラインスタイルの削減**  
   - 繰り返しのスタイルをクラスにまとめ、色・フォントサイズ・余白は CSS 変数またはクラスで管理する。

7. **[細部] マイクロインタラクション**  
   - ボタン・カードに hover 時の transition（opacity・transform・box-shadow）を追加する。  
   - 必要に応じて、FV やセクション見出しに軽い appear アニメーションを検討する。

8. **[細部] フォーム周りの見た目**  
   - 送信ボタンを CTA と同じ「行動」として扱い、サイズとコントラストを少し上げる。

## 唯一の良い点

**コピーと構成（不安→業者のリスク→元警察官の安心→強み→流れ→CTA）** という流れは、遺品整理LPとして論理的で、**コンテンツの順序は活かせる**。デザインとビジュアル階層を整えれば、同じ構成のまま「信頼感とモダンさ」を足しやすい。
