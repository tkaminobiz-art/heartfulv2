# ハートフルクリーン LP改善レポート ─ 実装状況

レポートの項目ごとに、**実装済み**・**一部対応**・**未対応**を整理しました。

---

## 実装済み

### 提案⑥ 固定CTAバー
- **SP時**: スクロールで表示される固定バー（電話＋LINE）を実装済み（`#floating`、200px以上スクロールで `.visible`）。
- **PC時**: 769px以上では非表示（ヘッダーに電話番号あり）。

### 提案⑧ FAQアコーディオン
- 質問クリックで回答を開閉するアコーディオンを実装済み（`.faq_q` → `aria-expanded` 切り替え、`.faq_a` の display 制御）。  
- 開閉の見た目（アイコン・罫線）をより分かりやすくする余地はあり。

### 提案⑦に近い セクションごとのビジュアルリズム
- 左右カラムを維持しつつ、**セクションごとに左・右カラムの背景色を変更**（`section-row--cream` / `cream2` / `white` / `gray` / `navy` / `pale`）してメリハリを付与済み。
- 余白トークン（`--space-24` 〜 `--space-120`）とセクション padding の統一を実施。

### その他、レポート前から実施済みの改善
- **視覚階層**: FVのh1を拡大（clamp 3rem〜4.2rem）、`section_title--primary` で重要見出しを強調。
- **カード・CTA**: `.card` の影・ホバー（lift＋shadow）、電話・LINE・フォーム送信ボタンのホバーを統一。
- **代表プロフィール**: 写真200px、`.profile-hero` で主役化。
- **お客様の声**: 1件目を `.testimonial--featured` で強調、カードスタイル適用。
- **セクション画像**: 中央カラム内に配置（フル幅やめ）。
- **受付時間の矛盾**: レポート指摘を受け、CTAブロックの「9:00-19:00」を **「9:00-22:00」** に修正し、ヘッダー・フッターの「7:00〜22:00」と整合させました。

## 一部対応（要フォロー）

### 提案① 配色の絞り込み
- **未実施**: メインをネイビー＋ゴールド/ベージュに絞り、オレンジはCTAのみ、赤排除、グリーンはLINEのみ──といった**トーン設計の全面変更**はまだ行っていません。
- 現状もオレンジ（見出し・ボタン・枠・テーブル等）、緑（LINE・電話）、赤（業者選びセクション）が併存しています。  
→ レポートの「2色＋アクセント」に合わせる場合は、CSS変数とクラスの一括変更が必要です。

### 提案② ファーストビューの全面刷新
- **一部**: h1の拡大・LINEボタンの強調・リードのトーンダウンは実施。  
- **未実施**: 全幅ヒーロー画像を背景にした構成や、「キャッチ1本＋CTA1つ」への大胆な削りは未実施。  
→ ヒーロー画像の素材とレイアウト方針の決定が必要です。

### 提案③ Before/After
- **画像**: パスは `./ハートフルクリーンサイト画像/before-after/ケース１〜４…` で指定済み。サーバー・ファイル配置によっては表示されない場合あり（パス・権限の確認が必要）。
- **スライダー**: 左右ドラッグで比較するUIや、間取り・作業時間・料金の併記は未実装。

### 提案④ お客様の声の強化
- **実施**: 1件目を大きく見せる・カード化。
- **未実施**: ★★★★★ の星アイコン、Google口コミの総合評価数値の大きくの表示、Googleウィジェット埋め込み。

### 提案⑤ ストックフォトの差し替え
- **未実施**: 自社撮影・オリジナル写真への差し替えは、撮影・素材準備が必要なためコード側では未対応。  
→ 画像差し替え用のファイル配置・パス対応は可能。
