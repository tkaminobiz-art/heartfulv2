# 道B LP 配置・バランス分析（ディレクター・デザイナー）

**日付**: 2026-03-13  
**対象**: `ihin-seiri-award-b.html`（PCフル画面表示）  
**課題**: オブジェクトが左寄せ・右寄せに分かれて見え、全体のバランスに違和感がある。

---

## 1. 現状の配置パターン

| エリア | 現状 | 要因 |
|--------|------|------|
| **ヘッダー** | 左：ブランド / 右：電話・CTA | grid 1fr auto → 意図的な左右分割（許容） |
| **ヒーロー** | 左：画像 / 右：テキスト・CTA | 2カラム grid 1.08fr 0.92fr。右カラムは左揃えで一貫あり |
| **信頼バー** | 左寄せ | justify-content: flex-start → 仕様どおり |
| **お悩み** | 左：バブル群 / 右：写真 | 2カラム。**右の写真に justify-content: flex-end** → 写真が右端に張り付く |
| **ご安心** | 左：バッジ・文・CTA / 右：画像 | **reassure-visual に justify-self: end** → 画像が右端に張り付く |
| **CTAバー** | テキスト左端・ボタン右端 | justify-content: space-between → **両端に要素が離れる** |
| **選ばれる理由** | 左：カード群 / 右：写真 | **reason-photo に justify-self: end** → 写真が右端に張り付く |
| **料金・お約束・支払い** | コンテナ内左揃え | 問題なし |
| **実績・口コミ・流れ・FAQ** | コンテナ内 | グリッドはあるが、全体はコンテナ幅に収まっている |
| **最終CTA** | 左：文 / 右：ボタン群 | grid 1.1fr 1fr → 再度強い左右分割 |
| **フッター** | 左揃え | 問題なし |

---

## 2. バランスが崩れる主因（設計観点）

1. **右カラムの「右端張り付け」**  
   `justify-self: end` / `justify-content: flex-end` により、右側の画像が常にコンテナ右端に寄る。  
   → 左ブロックと右ブロックが「中央で離れた二塊」に見え、余白の一貫性が失われる。

2. **CTAバーの両端配置**  
   `space-between` でテキストとボタンが画面幅いっぱいに離れる。  
   → 他セクションの「コンテナ内の塊」と比べて、一要素だけが極端に横に広がり、違和感につながる。

3. **見出しと本文の軸のゆれ**  
   見出し・リードは左揃えで統一されているが、2カラムで「左文・右画像」かつ画像が end だと、セクション単位で「左に重い・右に一塊」が強調され、ページ全体の軸がぶれて見える。

4. **テキストの折り返し**  
   リード文に max-width（例: 62ch）が無いと、大画面で1行が長くなり、ブロックの形が不安定になる。

---

## 3. 修正方針（デザイン仕様との整合）

- **デザイン仕様**: 「中央一点集約禁止」「左寄せ or 非対称 60:40」「コンテンツ幅を決め左右マージン 10〜15%」を維持する。
- **バランス**: 「左寄せ・非対称」は維持しつつ、**右カラムは「右端に張り付けない」**ようにする。  
  → 右側要素はグリッドセル内で start にし、左カラムとの間のガターを一定に保つ。
- **CTA**: テキストとボタンを「一つの塊」として扱い、左揃えでまとめる。必要なら折り返しで縦に並べる。
- **最終CTA**: 1カラム化（文→ボタンの縦並び）で、コンテナ内左揃えにし、左右の極端な分割を弱める。
- **テキスト**: リード・長文に `max-width: 62ch` をかけ、大画面でも行長を安定させる。

---

## 4. 実施した修正（実装）

- `.trouble-photo`: `justify-content: flex-end` を削除し `align-items: center` に → 写真を右カラムの先頭（ガター隣）に配置。
- `.reassure-visual`: `justify-self: end` を削除 → 画像を右カラムの先頭に配置。
- `.reason-photo`: `justify-self: end` を削除 → 同様に右カラム先頭に配置。
- `.cta-bar-inner`: `justify-content: space-between` を削除。`.cta-bar-text` に `max-width: 42ch` でテキストとボタンを一つの塊として左揃えに。
- `.final-cta-inner`: `grid-template-columns: 1fr`、`max-width: 640px` に変更。文の下にボタンを縦並びで左揃え。
- `.lead` に `max-width: 62ch`、`.section-head .h2` に `max-width: 32ch` を付与し、折り返しを安定化。
- `.section-head`: `text-align: left` を明示し、見出しの軸を統一。

以上で、**「左寄せ・非対称」は維持したまま、右への張り付けと CTA・最終CTAの極端な左右分割を抑え、全体のバランスを整える**。
