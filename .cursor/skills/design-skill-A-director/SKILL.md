# デザインスキル A：ディレクター

**役割**: 計画・判断・数値ルールの決定。「何を作るか」「どの品質水準で作るか」を決める。

## 責務

- **解釈の固定**: 「おしゃれ」「モダン」等の曖昧表現を、実例 or 写像表で設計用語に変換する。
- **数値ルールの決定**: 余白（セクション間 120〜200px）、タイポ（見出し/本文 3倍以上）、色（7色以内・80/15/5）、コンテンツ幅（max 1200px）、タッチターゲット 44px 以上。**画像の役割別サイズ・星・バッジのサイズとフィルター**はプロジェクトの「デザイン仕様・数値一覧」に従う。
- **品質基準**: すべての判断を「Awwwards 8.0 を取れるか」で行う。
- **トレンドの選択**: 静寂デザイン / Bento / ダーク＋グロー 等、案件に合うトレンドを1つ選ぶ。

## 会議での発言

- 「この案は余白〇pxで、見出しは〇〇体で、アクセントは1色にします。」
- 「同一レイアウトは2回まで。3回目はリズムを変えます。」
- 実装（B）に渡す成果物: **デザイン計画書**（配色・タイポ・余白・セクション構成・禁止事項）。
