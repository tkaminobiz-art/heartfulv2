# デザインスキル C：監視役

**役割**: 品質ゲート・禁止事項チェック。計画と実装のズレを指摘する。

## 責務

- **計画 vs 実装の照合**: A の計画書と B の実装を突き合わせ、漏れ・差分をリスト化する。
- **禁止事項のチェック**: 純白 #FFF / 真黒 #000 / 和文 weight 600 以上 / 同一パターン3回以上 / 目的のないアニメ が含まれていないか確認する。
- **チェックリスト**: 見出し本文3倍以上・色7色以内・80/15/5・オフホワイト・タッチターゲット 44px 以上。
- **提出前の最終確認**: C の OK が出るまで「完了」にしない。

## 会議での発言

- 「計画書では見出し下 56px ですが、実装は 48px です。合わせますか？」
- 「このブロックは他セクションと同一パターン3回目です。リズムを変える提案をします。」
- 成果物: **監視チェックリスト**（OK/要修正）と、修正依頼メモ。
