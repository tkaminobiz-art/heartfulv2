# LPの方向性 — 競合対比と「違和感」の言語化

**目的**: 「デザインは美しいが、LPとしてこのページだけで物を売る構成になっているか」「参考サイトのような色の使い方をするべきでは」という違和感を、競合のスクリーンショット・コード分析と照らして言語化し、正しい道かどうかを固める。

**作成日**: 2026-03-13

### 1-3. 競合に共通する「LPらしさ」（参考サイトの型）

- **色**: **オレンジ・黄** を「押す」「相談する」に使う。＝**行動色**がはっきりしている。
- **FV**: **料金の hint（最安値・無料見積・追加料金なし）** や **数字（相談数・評価）** を**最初から**出す。
- **CTA**: **電話番号** と **メール/相談ボタン** を **ヘッダー＋ページ内で繰り返し**。ボタンは**目立つ色・大きさ**。
- **密度**: 同じメッセージとCTAが何度も出てきて、「この1ページで決断してもらう」流れになっている。

＝ **「このページだけで物（＝相談・見積）を売る」＝ 転換重視のLP** の型。

## 2. 私たちのLP（ihin-seiri-award.html）がやっていること

| 観点 | 私たちの実装 |
|------|----------------|
| **色** | **緑（#2db742）** のみ。オレンジ・黄は使わない。オフホワイト・ベージュ・淡い緑で**落ち着き・信頼**。 |
| **FV** | 「**元警察官の代表が、最初から最後まで責任対応**」＋見積無料・キャンセル無料の**文言**。料金の数字（〇〇円から・最安値）は**出さない**。 |
| **CTA** | 電話・LINE・無料見積の3本はあるが、**緑で統一**し、ボタンは角丸12pxで**控えめ**。同じCTAが下でも出るが、**見た目のインパクトは弱い**。 |
| **情報の密度** | 余白を多く取り、セクションは「読む」構成。**ストーリー（元警察官・リスペクト・靴下エピソード）** が主で、**「今すぐ押せ」の繰り返し**は少ない。 |
| **トーン** | **信頼・誠実・丁寧**。「業界最安値」のような**比較・価格訴求**はしない。 |

＝ **「この会社を信頼してもらう・選んでもらう」＝ 信頼・ブランド重視のページ**。  
見た目も**きれいにまとまった、ホームページやブランドサイトのような1ページ**になっている。

## 3. 違和感の正体 — 言語化

あなたが感じている違和感は、次のように言い換えられると思います。

- **「参考サイト（競合）のような色の使い方をするべきでは」**  
  → 競合は **オレンジ・黄** で「相談する」「見積もりする」を**視覚的に強調**している。私たちは **緑だけ** で、**行動を促す色**が弱い。だから「LPとして押しが足りない」と感じる。

- **「きれいにまとまりすぎていて、このページだけで物を売る構成になっているか」**  
  → 私たちのページは **「読ませる・信頼させる」** 設計で、**「この1ページで問い合わせを取る」** 設計になっていない。  
  競合は **料金の hint・数字・繰り返すCTA・目立つボタン** で「このページで決断してほしい」と誘導している。  
  私たちは **余白が多く、ストーリーが主で、CTAは控えめ** なので、**ホームページ感（会社紹介・信頼構築）** が強く、**転換用LP感（今すぐ相談を取る）** が弱い。

- **一言でいうと**  
  **「美しいが、転換重視のLPの型（参考サイトの型）とは違う。信頼・ブランド重視の“きれいな1ページ”になっている。」**  
  という違和感。

## 4. 私たちは正しい道に進んでいるのか — 2つの道

### 道A: 「信頼・ブランド重視」のまま進む（現状の延長）

- **前提**: ハートフルクリーンの強みは **元警察官・リスペクト・丁寧さ**。競合と同じ「オレンジ・最安値・繰り返しCTA」にすると、**差別化が薄れる**。
- **狙い**: **信頼してから動いてもらう**。美しさと落ち着きで「この会社に頼みたい」と思ってもらい、そのうえで電話・LINE・見積に誘導する。
- **リスク**: 「LPとして押しが弱い」「このページだけで売れている感がしない」という違和感は、**あえて受け入れる**選択。転換率は**別施策（CTAの文言・配置の微調整、A/Bテスト）** で補う。

→ **「正しい道」とするなら**: 「私たちは競合と違う。転換LP型ではなく、**信頼ファーストのアワード級1ページ**で行く」と**方針として固める**必要がある。

### 道B: 「転換LPの要素」を足して、参考サイトに近づける

- **前提**: 「このページだけで物を売る」感を出したい。そのために、**競合がやっている要素の一部**を取り入れる。
- **具体例（取り入れ候補）**  
  - **色**: CTAボタンだけ **オレンジ** にする、または FV に **黄・オレンジの帯**（「見積無料・追加料金なし」）を1本入れる。＝ **行動色** を1つ足す。  
  - **FV**: 「**〇〇円から**」「**業界最安値**」は使わなくても、「**料金の目安はお見積りで**」の近くに **小さく「1K〜〇〇円」のような hint** を出せるなら出す。＝ **料金を想像しやすくする**。  
  - **CTAの見せ方**: 電話番号を **ヘッダーで大きく** する。同じ「無料相談・見積」ボタンを **オレンジまたは黄** で1本だけ「メインCTA」として強くする。  
  - **密度**: セクション間に **同じCTA（電話＋無料見積）** を、小さなバーやボタンで **もう1〜2回** 繰り返す。
- **注意**: やりすぎると「競合と同じテンプレ」になる。**色とCTAの強調だけ**に絞り、**余白・ストーリー・元警察官の軸**は残す。

→ **「正しい道」とするなら**: 「美しさは残しつつ、**転換LPの型を少し取り入れる**（行動色・FVの料金hint・CTAの強調・繰り返し）」と**方針として固める**。

## 5. 結論 — 固めるべきこと

- **違和感の正体**  
  「**転換重視のLP（参考サイトの型）**」と「**信頼・ブランド重視のきれいな1ページ（今の私たち）**」の**設計思想の差**。  
  色（オレンジ・黄の有無）、FVの訴求（料金・数字の有無）、CTAの強さと繰り返し、情報の密度、の4点で競合とずれている。

- **正しい道かどうか**  
  **どちらも「正しい」** が、**選ぶ必要がある**。  
  - **道A**: 違和感を「私たちは信頼ファーストで行く」と割り切る。  
  - **道B**: 違和感を解消するために、**行動色・FVの料金hint・CTAの強調・繰り返し**を**ほどよく**足す。

- **次のステップ**  
  あなたが **道Aか道Bのどちらに同意するか** を決める。  
  同意したうえで、「では道Bで進める。CTA色とFVのhintをこうする」など**具体的な実装**に落としていく。
