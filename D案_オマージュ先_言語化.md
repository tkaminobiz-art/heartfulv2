# D案 オマージュ先 — 完全言語化

**参照URL**：[こころテラス東海 LP](https://www.cocoroterrace-ihinseiri.com/lp/)  
**目的**：構造・トーン・コンポーネント・導線を言語化し、ハートフルクリーン向け D案 の再現設計に使う。

| 項目 | 言語化 |
|------|--------|
| **ページの役割** | 東海エリア（愛知・岐阜・三重）の遺品整理・生前整理の問い合わせ・見積り依頼を取る LP。電話と LINE（スピード見積もり）の二軸 CTA。 |
| **トーン** | 丁寧・フォーマル。「〜でございます」「〜させていただきます」を多用。権威（調査1位、資格、所属団体）と具体（料金表・事例・エリア一覧）で信頼を積む。 |
| **差別化の打ち出し** | 日本トレンドリサーチ「口コミ満足度第1位」「信頼・安心第1位」を FV 直下で提示。14の理由・料金の明示・買取・無料引き取りで「わかりやすさ」を強調。 |

## 2. 全体構成（セクション順・見出しレベル）

1. **ヘッダー**（固定 or 上部）  
   - 電話受付時間・一言メッセージ・「電話をかける」CTA・「見積り・ご依頼」→「スピード見積もり」（LINE）  
   - ナビ：ホーム／当社の特長／サービス内容・エリア／お客様の声・新着情報／費用のご案内／ご依頼の流れ／会社紹介（アンカーリンク想定）

2. **FV（ファーストビュー）**  
   - メインコピー（繰り返し）：**「なかなか出来ない遺品整理や生前整理。プロに任せてみませんか？」**  
   - 直下：**権威メッセージ**（2021年11月 日本トレンドリサーチ 東海エリア「口コミ満足度第1位」「信頼・安心第1位」＋感謝の挨拶文 2〜3 行）  
   - 対象地域の明示（愛知県・岐阜県・三重県）

3. **こんなことにお困りなら当社にご相談ください**  
   - 見出し H3  
   - 箇条書き（9項目）：訃報・老人ホーム入所・家具移動・遠方・実家整理・買取希望・退去・生前整理・管理アパート失踪 など、**悩みの言語化**のみ。CTA はなし。

4. **当社が選ばれる 14の理由**  
   - 見出し：「こころテラスが選ばれる14の理由」  
   - 01〜14 の番号付きカード。各カード：**短いタイトル**（例：01貴重品、重要品も任せて安心です。）＋**説明 2〜4 行**。  
   - 内容：貴重品返却・迅速作業・買取・金額提案・引越し・終活断捨離・捜索・リサイクル家電許可・配送・終了報告・廃棄業者紹介・供養・ハウスクリーニング・データ消去。

5. **サービス内容**  
   - 見出し H3「サービス内容」  
   - カード形式 7 種：遺品・生前空き家整理／ゴミ屋敷・汚部屋清掃／特殊清掃／簡易引越し／買取／リサイクル家電廃棄代行／貴重・必要品発送代行。  
   - 各カード：**サービス名**＋**1〜2 行の説明**。

6. **サービスエリア**  
   - 見出し「サービスエリア」＋リード 1 行（名古屋中心・東海）。  
   - 愛知県・岐阜県・三重県の**市町村名を列挙**（テキストブロック）。

7. **お客様の声**  
   - 見出し「お客様の声」  
   - **事例カード**：タグ（遺品整理／遺品整理 ゴミ屋敷 等）＋**見出し**（例：岐阜県岐阜市 4LDK 115,000円）＋**本文**（依頼内容・経過・お客様の声の引用）。  
   - 最後に「もっと見る」→ブログ or 一覧ページへのリンク。

8. **新着情報**  
   - 見出し「新着情報」  
   - 日付＋タイトル（リンク）＋概要 1〜2 行。複数件リスト。

9. **費用のご案内**  
   - 見出し「費用のご案内」  
   - **キャッチ**：「わかりやすい料金設定」「１K35,000円～」「物量にあわせた料金設定なので通常作業では追加料金は発生しません」  
   - **間取り別料金表**（1R〜3LDK、合計金額の目安）  
   - 注意書き（家電リサイクル・オプション別、廃棄は自治体により異なる）  
   - **清掃オプション料金**（トイレ・浴室・キッチン・床・窓など単価またはセット）  
   - **不用品買取・無料引き取り**：対象品の箇条書き。  
   - **お支払方法**：現金・振込（後払い）・クレジットカード。前払い条件の説明。

10. **ご依頼の流れ**  
    - 見出し「ご依頼の流れ」  
    - **01〜05 のステップ**：お問い合わせ→無料見積り→見積・成約→契約・支払い→作業完了。各 1〜3 行。  
    - 補足ブロック：「貴重品は必ずお引き渡し」「廃棄物の処理は法令順守」＋注意 1 行。

11. **よくあるお問い合わせ**  
    - 見出し「よくあるお問い合わせ」  
    - **Q&A 形式**（Q と A）。急な相談・悪臭・土日祝・費用の決め方・遠方・見積以上請求・汚染物廃棄・遺品以外の作業 など 8 問前後。

12. **無料お見積りフォーム**  
    - 見出し「無料お見積りフォーム」  
    - リード：「訪問お見積り・ご相談は無料。返信にお時間かかる場合あり、お急ぎは電話を。」  
    - **フォーム項目**：送信内容／ご依頼内容／お名前・かな／電話番号／メール・再入力／連絡方法／住所／表札／間取り／建物種類／見積希望日・希望時間（第1〜3）／メッセージ本文。  
    - プライバシー同意・スパム防止チェック・送信ボタン。

13. **会社紹介**  
    - 見出し「会社紹介」  
    - **代表写真・名前**（代表取締役 ○○）  
    - **代表挨拶**：個人ストーリー（父の死→遺品整理の経験→業界リポート→M&Aを機に転身→苦難と成長→「お客様を想う」）4〜6 行。

14. **会社概要**  
    - 見出し「会社概要」  
    - **表形式**：会社名・代表・設立・資本金・業務内容・本社・営業所・フリーアクセス・営業時間・取得資格・所属団体・取引銀行 など。

15. **プライバシーポリシー／免責事項**  
    - 見出しで区切り。プライバシーポリシー（収集・使用・安全管理・Cookie・SSL）・免責事項 3 条。

16. **情報セキュリティ基本方針**  
    - 見出し＋5 項目の短文。

17. **フッター**  
    - 電話番号・受付時間・一言・対象地域・会社名・住所・関連HP・©。

## 3. CTA と導線の言語化

| 場所 | 要素 | 文言・役割 |
|------|------|------------|
| ヘッダー | 電話 | 「電話をかける」＋tel:リンク。補足「電話受付 7:00～24:00」「遺品整理サイトを見ましたとお知らせ下さい」 |
| ヘッダー | LINE | 「見積り・ご依頼」→「スピード見積もり」（LINE URL）。メインの低ハードル CTA。 |
| FV | なし | 本文のみ。権威メッセージで信頼を置かせ、下スクロールへ誘導。 |
| お客様の声 | 出口 | 「もっと見る」→ブログ／一覧。 |
| 費用・流れの後 | フォーム | 無料お見積りフォームへアンカー。 |
| フォーム上 | 電話の促し | 「お急ぎの方は直接お電話で」 |

**導線の特徴**：FV で「問い合わせ・見積り」の意欲を高め、14の理由・サービス・事例・料金で納得を積み、最後にフォーム or 電話／LINE。ヘッダーは常時「電話」「LINE」を出し続ける。

## 4. デザイン・コンポーネントの言語化（推測含む）

| 要素 | 言語化 |
|------|--------|
| **レイアウト** | 1カラム中心。セクションは白背景＋見出し＋本文／カード群。ヘッダーはコンパクトで CTA 2 本立て。 |
| **見出し** | H2 でセクション名。H3 でサブ（「こんなことにお困りなら」等）。番号付き（01〜14、01〜05）で「理由」「流れ」を整理。 |
| **カード** | タイトル＋本文。枠線 or シャドウでブロック化。14の理由は 2 列 or 1 列の繰り返し。 |
| **表** | 間取り〜料金、会社概要は表（thead／tbody）。 |
| **リスト** | 悩み・買取対象・無料引き取り対象・Q&A は箇条書き or 定義リスト。 |
| **フォーム** | ラベル・必須・選択（送信内容・依頼内容・間取り・建物・連絡方法・日付）・テキストエリア・同意・送信。 |
| **色・トーン（推測）** | 信頼感のある落ち着いた色。CTA は電話・LINE で色分け可能。 |

※ 実際の色・フォント・余白・SP レイアウトは実機確認 or スクショで補足する。

## 6. オマージュの範囲（再現する／しない）

| 再現する | 再現しない（ハートフル用に変える） |
|----------|------------------------------------|
| セクションの**順序**（ヘッダー→FV→悩み→理由→サービス→エリア→声→新着→費用→流れ→FAQ→フォーム→会社紹介→概要→フッター） | 文言はすべてハートフルクリーン・奈良向けに**オリジナル** |
| **14の理由**の「数」とカード形式（中身は差し替え） | 新着情報は**ブログがあればリンク、なければ削除 or 簡略** |
| **料金表・清掃オプション・買取・支払い**の**構成** | フォームは**実際の送信先・項目**に合わせて実装 |
| **ご依頼の流れ 01〜05** のステップ構成 | プライバシー・免責・セキュリティ方針は**自社用**に差し替え |
| **Q&A** の形式と問いのカテゴリ | エリアは**市町村一覧**を奈良・近畿用に作成 |
