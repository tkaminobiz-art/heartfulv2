# ハートフルクリーン LP — Webデザインスキル診断（静寂モダン基準）

**診断日**: 2026年  
**スキル参照**: `.cursor/skills/web-design/SKILL.md` ＋ patterns.md / trends.md  
**ユーザー要望**: ミニマルでモダン、パレット見直し

---

## 設計宣言（目指す姿）

```
【設計宣言】
アーキタイプ: 静寂モダン（Seijaku Modern）
トレンド: 静寂デザイン（Exhale）、ノイズテクスチャは控えめに維持可
配色: オフホワイト / ソフトブラック / アクセント1色（彩度を落とした緑）
フォント: 見出し 明朝（Shippori Mincho または Noto Serif JP）+ 本文 Noto Sans JP
余白感: ゆったり（セクション間 120〜200px、コンテンツ幅 960px 以下）
アニメーション: 控えめ（フェードインのみ、ホバーは opacity）
一言: 「余白が呼吸している。信頼と落ち着きを、装飾に頼らず伝える。」
```

## 1. 配色（パレット）の問題点

### 現状の変数

| 変数 | 現状値 | 用途 |
|------|--------|------|
| --primary | #2D5A3D | 緑（メイン） |
| --primary-dark | #1E3D2A | ヘッダー・濃い緑 |
| --primary-light | #3A6B4A | （未使用に近い） |
| --accent | #E67E22 | オレンジ（CTA・バッジ） |
| --accent-hover | #D35400 | オレンジ hover |
| --bg | #FAFAF8 | 背景 |
| --bg-alt | #F0F4F1 | セクション交互 |
| --bg-dark | #2D5A3D | ダークセクション |
| --text | #1C1C1C | 本文 |
| --text-light | #666666 | サブ |
| --badge-bg | #EDF5F0 | バッジ背景 |

**スキル（静寂モダン）の制約**:  
「色数の上限: 背景系2色 + テキスト系2色 + **アクセント1色** = 最大5色」  
「アクセント: **1色のみ**。彩度を落とした色（例: #B8860B, **#5C6B5E**, #8B6F5E）」

### 問題一覧

| # | 問題 | スキル上の理由 |
|---|------|----------------|
| 1 | **アクセントが2系統**（緑＋オレンジ） | 静寂モダンはアクセント1色。緑とオレンジの併用は「信頼コーポレート」や「温もりナチュラル」のパターン |
| 2 | **オレンジ #E67E22 が高彩度** | 静寂モダンは「彩度を落とした色」。このオレンジは目立ちすぎて「静かさ」を壊す |
| 3 | **primary が3段階**（light / 通常 / dark） | 役割が重複。ヘッダー用の暗い色は「アクセントの暗い版」1つで足りる |
| 4 | **badge-bg が独立** | 背景系に含めると色数オーバー。アクセントの極薄版（例: アクセント 5% opacity）で代替可能 |
| 5 | **--bg-alt #F0F4F1** | 緑がかった灰。静寂モダン推奨は #F5F3EF（ニュートラルなオフホワイト）。統一した方が「間」がきれい |

## 2. パレット見直し案（静寂モダン準拠）

### 案A: 緑1色に統一（ミニマル・信頼）

| 役割 | 変数名 | 新値 | 備考 |
|------|--------|------|------|
| 背景 | --bg | #FAFAF8 | 現状維持 |
| 背景alt | --bg-alt | #F5F3EF | ニュートラルに |
| テキスト | --text | #1A1A1A | 純黒避け |
| テキストサブ | --text-muted | #8A8480 | 静寂モダン推奨 |
| アクセント | --accent | #5C6B5E | 彩度落とし緑（1色のみ） |
| アクセント・濃（ヘッダー等） | --accent-dark | #2C3D32 | 同じ色相の暗い版 |
| ボーダー | --border | #E8E4DE | 静寂モダン推奨 |

- **CTA**: オレンジをやめ、`--accent` または `--accent-dark` のボタンに統一。
- **LINE**: 緑 #06C755 はブランド色のため維持可（「アクセント」ではなく「外部サービス色」として扱う）。

### 案B: 緑＋くすみ金（上品な差し色）

- 上記に加え、**アクセントの差し色**を1つだけ: `--accent-secondary: #8B6F5E`（くすみ茶）または `#B8860B`（ダークゴールド）。
- 使用箇所は「見出しのラベル」「区切り線」など**ごく限定的**にし、メインのCTAは緑のまま。

### 推奨

- **まずは案A（緑1色）で統一**し、余白・タイポ・影の削減と合わせて「ミニマルでモダン」を達成する。
- 必要なら後から案Bで accent-secondary を「調味料」として追加。

## 3. その他の問題点（スキル基準）

### 3.1 タイポグラフィ

| 問題 | 現状 | 静寂モダン推奨 |
|------|------|----------------|
| 見出しサイズ | clamp(28px, 3.5vw, **44px**) | clamp(1.8rem, 3vw, **2.8rem**) ≒ 約29px〜45pxだが「大きすぎない」が原則なので **2.5rem 前後**が無難 |
| 見出しフォント | Noto Serif JP | Shippori Mincho が推奨（和の静寂）。Noto Serif JP でも可 |
| 本文 line-height | 1.9 | 1.8〜2.0 で問題なし |
| フォントウェイト | 複数（400, 500, 600） | 見出し・本文・ラベルの3段階に収める |

### 3.2 余白・レイアウト

| 問題 | 現状 | 推奨 |
|------|------|------|
| コンテンツ幅 | **1100px** | **960px 以下**（静寂モダン: 800〜960px） |
| セクション間 | 100〜180px など | 120〜200px で統一感を持たせる |
| 左右パディング | clamp(20px, 5vw, 80px) | 5〜8% が目安 |

### 3.3 影・装飾（引き算）

| 問題 | 現状 | 推奨 |
|------|------|------|
| ボタン .btn--primary | box-shadow + **::after の blur 擬似要素** | 静寂モダン「影の多用禁止」。**影と ::after は削除**し、単色ボタンに |
| ボタン hover | transform: translateY(-1px) + box-shadow 増 | **opacity 変化のみ**（スキル: ホバーで transform 使わない） |
| カード | box-shadow: var(--shadow-lg) | 必要最小限に。**border のみ**でも可 |
| オープニング | radial-gradient の glow | 削除または極めて控えめに（単色背景で十分） |
| ノイズテクスチャ | 全ページ固定 | 静寂モダンでは「調味料」。**opacity 0.02〜0.03** に下げるか、削除 |

### 3.4 角丸

| 問題 | 現状 | 推奨 |
|------|------|------|
| バリエーション | 4, 6, 8, 10, 12px + 100px（ピル） | **統一**: 4px（ボタン・入力）と **8px**（カード・画像）の2段階に絞る |
| 静寂モダン | — | 「角丸の大きいカード(16px超)禁止」→ 12px は許容だが、**8px で統一**するとよりミニマル |

### 3.5 アニメーション

| 問題 | 現状 | 推奨 |
|------|------|------|
| フェードイン | translateY + opacity、duration 様々 | **translateY は最大 16px**（現状も概ねOK）、duration を **1.0〜1.2s** に揃える |
| ホバー | ボタン・カードで translateY(-1px) 等 | **opacity のみ**に変更 |
| オープニング | 複数 keyframes（glow, circle-shrink 等） | シンプルに「ロゴ＋フェード」に縮小するか、そのままなら「重くしすぎない」程度に |

### 3.6 禁止事項チェック（静寂モダン）

- **グラデーション背景**: オープニングの glow、サービスカードの overlay に使用 → **削除または単色に**
- **影の多用**: ボタン・カード・ヘッダー → **削減**
- **アイコン装飾**: セクションラベルの SVG 等 → 必要最小限なら可。**バッジの横並びは「質」で差をつける**（スキル）
- **太字の濫用**: 現状 500, 600 が散在 → **見出し・CTA に限定**

## 5. パレット変更時の置き換え一覧（実装用）

```css
/* 現状 → 新（案A） */
--primary: #2D5A3D       → --accent: #5C6B5E;
--primary-dark: #1E3D2A  → --accent-dark: #2C3D32;
--primary-light          → 削除（必要なときは accent の lighten）
--accent: #E67E22        → 削除（CTA は --accent または --accent-dark に）
--accent-hover           → --accent の hover は opacity で対応
--bg-alt: #F0F4F1        → #F5F3EF
--text-light: #666666    → --text-muted: #8A8480
--badge-bg               → rgba(92,107,94,0.08) など accent の薄い版
```

## 6. 実行ログ（実施済み）

- **パレット**: 案Aを適用。`--accent` を #5C6B5E、`--accent-dark` を #2C3D32 に。オレンジ廃止。`--bg-alt` を #F5F3EF、`--text-light` を #8A8480、`--badge-bg` を accent の薄い版に変更。
- **コンテンツ幅**: 1100px → 960px。角丸は 8px に統一（card-sm/md/lg）。
- **ボタン**: `.btn--primary` の影・`::after` 削除。ホバーは `opacity` のみ（transform 削除）。primary / outline / white / phone を緑系に統一。
- **FV・カード**: FV写真は `box-shadow` をやめ `border` のみ。サービスカードのオーバーレイをグラデーションから単色 `rgba(44,61,50,0.88)` に変更。
- **ノイズ**: opacity 0.048 → 0.022。
- **オープニング**: `opening__glow` を `display:none` にし、背景は `var(--accent-dark)` 単色に。
- **ヘッダー**: 背景を `var(--accent-dark)` に。ロゴに `filter: brightness(0) invert(1)` を指定し暗いヘッダー上で白表示に。
- **見出し**: `.section-heading` の最大サイズを 44px → 32px に縮小。
- **CTAバンドの装飾**: オレンジの radial-gradient を緑系 `rgba(92,107,94,0.06)` に変更。

以上を反映済み。
