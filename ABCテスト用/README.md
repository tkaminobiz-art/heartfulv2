# ABCテスト用 — 案別コードの保管

## 現在のトップページ

- **メイン（ルート）**: ルートの `index.html` は **D案** へのリダイレクトです。アクセスすると `ihin-seiri-d.html`（刷新中の遺品整理LP）に遷移します。
- **D案の実体**: ルートの `ihin-seiri-d.html` が、現在刷新している提案（メインページ）です。

## 方針

- **A案・B案・C案のデータは削除しません。** すべてこのフォルダまたはルートに残しています。
- トップをA案に戻したい場合は、下記「復旧手順」を参照してください。

## フォルダ構成

| 場所 | 内容 |
|------|------|
| **ルート index.html** | D案へのリダイレクト（0秒で ihin-seiri-d.html へ） |
| **ルート ihin-seiri-d.html** | D案（刷新中メインLP） |
| **ルート ihin-seiri.html** | C案（本番・ライティング用として使用していた案） |
| **ルート index_b.html** | B案（参照用に残す） |
| **A案/index.html** | 旧トップページの内容のコピー（リダイレクトに差し替える前の index.html） |
| **B案/index_b.html** | B案のコピー |

## 復旧手順（トップをA案に戻す場合）

1. ルートの `index.html` を、このフォルダの `A案/index.html` で上書きする。
   ```bash
   cp ABCテスト用/A案/index.html index.html
   ```
2. これでトップが再びA案の内容で表示されます。D案は `ihin-seiri-d.html` のURLで引き続き利用可能です。

## 注意

- A案・B案のHTMLは、画像・CSSなどの相対パスが**ルート基準**のままです。このフォルダ内だけで表示する場合は、パスの調整が必要になることがあります。
- 採点・比較用の資料はプロジェクトルートの `ABCA案_採点.md` を参照してください。
