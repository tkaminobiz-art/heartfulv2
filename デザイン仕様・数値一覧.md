# デザイン仕様・数値一覧（プロジェクト参照用）

実装（B）・監視（C）はこの一覧を「条件の正」として参照する。ディレクター（A）は計画書にこの条件を満たすことを明記する。

**上位参照**: web-design-creation Part A、design-system Part 1、image-usage-policy。ここに書いていない項目はそれらに準じる。

---

## 1. 余白

| 項目 | 条件 |
|------|------|
| セクション間 | 120〜200px |
| 関連要素間 | 8〜16px |
| 非関連要素間 | 48〜80px |
| コンテンツ左右マージン | 画面幅の 10〜15% |
| カード内 padding | 32〜48px |
| 見出し | 見出しの上 > 見出しの下 |

### レイアウト — 「真ん中にポツンと」禁止

- **チープ＝全部中央揃え / プレミアム＝非対称**（web-design-creation 準拠）。セクション内のコンテンツを **中央一点に集約しない**。
- 信頼バー・見出しブロック・カード群などは **左寄せ** または **非対称（例: 55:45, 60:40）** で配置。`justify-content: center` で横方向に中央寄せするレイアウトは避ける。
- コンテンツ幅（max-width）を決め、**左右マージン（画面幅の 10〜15%）** を取ったうえで、その中で **左寄せ** または意図した非対称にする。余白と密度のコントラストをはっきりさせる。

## 2. タイポグラフィ

| 要素 | サイズ | weight | 備考 |
|------|--------|--------|------|
| 見出し h1 | 48〜80px（clamp 推奨） | 300〜400 | 見出し/本文比 **3倍以上** を満たすこと |
| 見出し h2 | 32〜48px | 400〜500 |
| 小見出し h3 | 20〜28px | 400〜500 |
| 本文 | **16px**（15〜17px の範囲） | 400 | 和文は **weight 500 以下** |
| リード・サブ | 14〜18px | 400 |
| キャプション | 12〜13px | 400 |
| 数字・価格 | 欧文フォント推奨 | 300〜400 | tabular-nums |

- 和文 **font-weight: 600 以上禁止**。
- letter-spacing・line-height は design-system の具体スケールを参照してよい。

---

## 3. 配色

| 項目 | 条件 |
|------|------|
| 使用比率 | ニュートラル **80%** / メイン **15%** / アクセント **5%** |
| 背景 | オフホワイト（#FAFAF7, #F5F5F0）。純白 #FFFFFF 禁止 |
| テキスト | ソフトブラック（#1C1C1C, #222）。真黒 #000 禁止 |
| アクセント | **1色のみ**。2色以上は散漫になる |
| 色数 | 全体で **7色以内** |

---

## 4. 画像 — 役割別サイズ・扱い

| 役割 | サイズ・比率 | 備考 |
|------|--------------|------|
| **FV・ヒーロー主役画像** | 左 or 中央を **フル表示**（100vh またはカラムいっぱい）。小さくしない。object-fit: cover | 主役は「大きく・画面を占める」が原則 |
| カード内サムネイル | 高さ 160〜280px 程度、幅はグリッドに合わせる。aspect-ratio 統一 | 3:2 または 4:3 で統一 |
| 一覧・グリッドの小画像 | 100〜160px 程度。object-fit: cover、border-radius 8px |  |
| アイコン・記号 | 24〜44px（タッチする場合は **44px 以上**） |  |
| ロゴ | ヘッダー 100〜130px 幅程度。他は文脈に応じる |  |

- アスペクト比: 3:2 または 4:3 で**サイト内統一**。
- グラデーションのプレースホルダーは公開で使わない。AI生成の人物・完璧すぎるストックは避ける。

---

## 5. 装飾要素 — 星・評価・バッジ

| 項目 | 条件 |
|------|------|
| **星（★）・評価表示** | 表示サイズ **20〜28px**。タップ可能な場合はタッチ領域 44px 以上を確保 |
| フィルター | 星は**彩度を落としすぎない**（金・黄・オレンジはブランドに合わせる）。drop-shadow は控えめ（0 0 4〜8px、opacity 0.2〜0.4 程度）。過度の blur は使わない |
| バッジ・ラベル | フォント 12〜14px、padding 4px 12px 程度。角丸 4〜8px。同一スタイルを 3 回以上連続で使わない |

- 評価は「Googleで★5.0」など**出所を明示**する（reviews-google ルール準拠）。

---

## 6. フォントスタック（推奨）

- **和文**: Noto Sans JP / Zen Kaku Gothic New / ヒラギノ角ゴ 等、weight 400〜500 で統一。太字は 500 まで。
- **欧文・数字**: 和文と相性のよいサンセリフ（Inter, system-ui）＋ tabular-nums。
- 計画書で 1 セット決めたら、その案件ではそれに従う。

---

## 7. アニメーション・効果

| 項目 | 条件 |
|------|------|
|  duration | 400〜800ms |
| easing | cubic-bezier(0.16, 1, 0.3, 1) 等 |
| 目的のないアニメ | 禁止。削除する |
| 同一パターン | **2回まで**。3回目はリズムを変える |
| ノイズ・オーバーレイ | opacity 0.03 程度。 pointer-events: none |

---

## 8. 禁止事項（チェック用）

- 純白背景 #FFFFFF / 真黒 #000000
- 和文 font-weight 600 以上
- border-radius 999px（ピル型）
- **セクション内コンテンツを「真ん中にポツンと」中央一点に集約**（横方向の中央揃えのみで構成しない）
- 同一レイアウトパターン 3 回以上
- 均等グリッド 2 セクション連続
- グラデーションのプレースホルダーを公開で使用
- 目的のないアニメーション
