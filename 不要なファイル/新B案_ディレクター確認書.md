## 4. デザイン計画書サマリ（実装Bに渡す数値一覧）

実装時に参照する**配色・タイポ・余白**の確定値。

### 配色（HCブランド・案2）

| 用途 | 値 | 備考 |
|------|-----|------|
| メイン（ボタン・見出し・アクセント） | `#5C6B5E` | HC 緑 |
| メイン暗（ホバー・フッター背景等） | `#2C3D32` | 既存 index の --primary-dark 相当 |
| 背景ベース | `#FAFAF8` | オフホワイト（純白禁止） |
| 背景サブ（セクション交互） | `#F5F3EF` | 既存 --bg-alt 相当 |
| テキスト | `#1A1A1A` | ソフトブラック（真黒禁止） |
| テキスト補助 | `#8A8480` | キャプション・サブ |
| ヒーロー背景シェイプ | `#5C6B5E` の 0.9〜1 不透明度 | カーブ部分。青ではなく緑で統一。 |
| 電話ブロック・信頼の補助 | `#5C6B5E` または `#2C3D32` | 青は使わず緑系。 |
| CTAボタン（無料お見積り） | `#C4A84A` 程度の黄〜ゴールド | 目立つが品格あり。または HC 緑のまま。 |
| 追従CTA・ページトップ | 黄＋水色円は HC 緑＋薄緑に変更可 | プロスタッフの黄/水色を緑トーンで再解釈。 |
| 4カードテーマ | 緑のトーン差 4種（例: #5C6B5E, #6B7B6E, #4A5B4E, #7A8B7E） | 同一色相で明度・彩度を少しずつ変える。 |

### タイポ

| 数字 | Inter 等、`tabular-nums`。 |

### 余白・レイアウト

| 項目 | 値 |
|------|-----|
| セクション間 | 144px（9rem）を基準。リズムを変える箇所は 96px / 192px。 |

## 5. 実装GOの条件（実装中に守ること）

1. **上半分（§1〜§11）**: 完全言語化とスクショに**忠実**に再現する。ヒーロー・リード・認定・CTAバンド・4カード・メディア・対応エリアの順とレイアウトを崩さない。
2. **下半分（§12〜§18）**: 完全言語化の文言と実装計画の表に従う。実機スクショがないため、一般的なLPのベストプラクティス（01〜06 円形バッジ、カード横並び、アコーディオンFAQ）で構成し、**文言とセクション順**をプロスタッフに合わせる。
3. **配色**: 上記「4. デザイン計画書サマリ」の数値を使う。不明点は HC 既存 index.html の `:root` を優先。
4. **画像**: Part 6 のマッピング表に従う。不足分は Part 6 の「不足・要検討」の対応案または placeholder。
5. **電話・料金**: 確定次第、index-b-new.html 内を一括置換。確定前は index.html の表記を流用してよい。

## 6. ディレクター判定の要約

- **オマージュは「条件付きでできる」**。
- **理由**: トップ〜対応エリアまでは言語化・スクショ・コピー・デザイン・画像のすべてが揃っている。§12以降は言語化と実装計画のみだが、セクション構成・文言は確定しており、「再現度のばらつき」は実装後のスクショ比較で吸収可能である。
- **条件**: 上記「5. 実装GOの条件」を守り、配色は本確認書のサマリで統一する。電話・料金はクライアント確認前提で placeholder または既存表記で進める。
