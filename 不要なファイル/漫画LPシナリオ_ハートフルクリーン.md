# ハートフルクリーン　漫画LPシナリオ
## 「あの部屋の続きを、誰かがやらなければ。」

**制作用途：** ランディングページ（LP）掲載用漫画
**想定読者：** 遺品整理・特殊清掃を検討中の遺族 / ケアマネ経由の高齢者家族 / 不動産管理会社
**トーン：** 感動系・実話ベース・温かみのある絵柄（リアル寄りよりも、やや丸みのあるタッチ推奨）
**総ページ数目安：** 8〜10ページ（各ページ3〜4コマ）

---

## ■ 登場人物

| キャラクター | 設定 |
|---|---|
| **藤田 宗一郎（主人公）** | 元大阪府警の警察官。28歳。誠実で真面目、不正が大嫌い。現在はハートフルクリーン代表。 |
| **田中 和子（依頼人）** | 60代の女性。一人暮らしの母を突然亡くし、遠方から実家へ。片付けの方法が分からず途方に暮れている。 |
| **上司（警察時代）** | 藤田を見守る50代のベテラン刑事。温かみのある人物。 |

---

## ■ シナリオ全体の構成

```
【第1章】警察官・藤田の転機（過去回想）
→ 読者に「なぜこの仕事をしているのか」を刷り込む

【第2章】依頼人・田中和子の現実
→ 読者が「自分ごと化」できる悩みを描く

【第3章】出会いと解決
→ ハートフルクリーンのサービスが自然に紹介される

【エンディング】
→ 感動の締めくくりとCTA（問い合わせへの誘導）
```

## ■ 制作ディレクション上のポイント

### 絵柄の指示（画像生成AIへのプロンプトに活かせる方向性）
- 全体的に温かみのある、やや丸みのある線画タッチ
- 背景は省略気味でもOK（感情表現を優先）
- 藤田は「誠実そうな30代男性」。細かすぎる描写より「頼れるお兄さん」的な雰囲気
- 和子は「普通のお母さん」。特別な美化は不要。共感重視
- カラーはモノクロ（またはセピア）ベースでも可。感動系のトーン

### 各ページの感情設計
| ページ | ねらう読者の感情 |
|---|---|
| 1〜2 | 「知らなかった。こういう現場があるんだ」（問題認識） |
| 3〜4 | 「この人は本物だ」（代表への信頼） |
| 5〜6 | 「これ、自分のことだ」（共感・自分ごと化） |
| 7 | 「安心して頼めそう」（不安解消） |
| 8〜9 | 「感動・涙」→「問い合わせたい」（行動意欲） |

### 重要：漫画LP配置の推奨順序
1. FV（ファーストビュー）→ キャッチコピー
2. **漫画（ここ）** ← 読者を感情移入させる
3. サービス詳細・料金表
4. お客様の声
5. CTA（問い合わせボタン）
