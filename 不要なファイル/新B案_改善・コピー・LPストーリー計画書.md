# 新B案：改善点発見・コピー・LPストーリー ディスカッション＆計画書

**作成日**: 2026年3月  
**目的**: ヒーローコピー廃止と第三者目線の強化、LP全体の構成・コピーのブラッシュアップ。デザイン・LP専門・コピーライティングの3視点で改善点を発見し、計画を立ててから実装する。

---

## Part 1：改善点の発見

### 1-1. ヒーローセクション（現状の問題）

| 項目 | 現状 | 問題 |
|------|------|------|
| **実績表記** | 「おかげさまで実績 多数 件」「奈良・関西エリアでご依頼多数」 | 抽象的。「多数」は数字でないため信頼性が弱い。実績を前面に出すなら具体数値か、別の差別化にすべき。 |
| **キャッチ** | 「家族によりそう 遺品・生前整理」 | 情緒的だが、**ベネフィットが曖昧**。LP-OSの「FVはベネフィット＋具体的数字」に反する。 |
| **料金表記** | 「1R ご相談のうえお見積り 税込」 | 情報量が少なく、アンカリング効果も弱い。 |
| **締め** | 「Googleの口コミ ★5.0 ご依頼主様にご満足いただいています。」 | **企業主語・売り手目線**。「いただいています」は自社の言い回しで、第三者の声に見えない。 |

**結論**: ユーザー指摘どおり「弱い」ため**廃止**し、**第三者目線の一言**をメインに据える。

---

### 1-2. ユーザー提案の整理

- **メイン例**: 「**元警察官が代表の会社なので安心してお任せ頂いています！**」
  - **第三者目線**（お客様の声としての文体）
  - 「ください」ではなく「頂いています」＝すでに依頼した人の証言
  - **権威性**（元警察官）と**安心**を一発で伝える → ウィンザー効果・権威性の活用

この方向性を**ヒーローの核**とし、LP全体のストーリーと整合させる。

---

### 1-3. LP全体で見た改善点（3スキル視点）

#### A. コピーライティング視点

| セクション | 改善点 |
|------------|--------|
| **ヒーロー** | 上記の通り。第三者証言をメインにし、実績「多数」・「ご満足いただいています」は廃止または後方へ。 |
| **リード（b-lead）** | 「よかったねと、家族で満足できる整理です。」→ 良いが、その前に**問題の言語化**（PASのP）を短く入れると共感が増す。 |
| **認定2ブロック** | 「信頼の体制」の「曲がったことが嫌いな性分」は良い。Google口コミ側は**具体的な一声**を1文引用すると信頼が増す。 |
| **CTA周辺** | 「どうぞハートフルクリーンにご相談ください」は企業主語。**次の一手を1つ**にし、心理的ハードルを下げる文言（「電話1本で」「見積りだけでも」）を足す。 |
| **お客様の声** | 既存のGoogle口コミ2件は良い。**「元警察官だから安心した」系**の一声を1件追加すると、ヒーローと一貫する。 |

#### B. LP（構成・CVR）視点

| 項目 | 改善点 |
|------|--------|
| **結論ファースト** | FVで「何が得られるか」「なぜ信頼できるか」を3秒で伝える。現状は「遺品・生前整理」というサービス名が先で、**ベネフィットが後回し**。 |
| **ストーリー順** | LP-OSの推奨: Interest → Problem → Evidence → Empathy → Authority → Action。現状は「共感（リード）→ 信頼（認定）→ CTA」で、**Problemの明示**が弱い。 |
| **権威性の位置** | 「元警察官」は認定ブロックとお客様の声にあるが、**FV（ヒーロー）にない**。FVに第三者目線で入れると離脱率低減が期待できる。 |
| **数値・固有名詞** | 「追加料金なし」「★5.0」は良い。可能なら「〇〇件」「奈良で〇年」など**具体数値**を1つでも入れると説得力が増す。 |

#### C. デザイン視点

| 項目 | 改善点 |
|------|--------|
| **ヒーロー情報量** | 廃止する「実績多数」「1R税込」「ご満足いただいています」を削ることで、**テキスト量が減り視覚がスッキリ**する。 |
| **階層** | メインコピー（第三者証言）を**大きく1本**にし、サブで電話番号・CTAを置く構成にすると、Zの法則に沿う。 |
| **リズム** | ヒーローを短くする分、その次のリード・認定で「共感→根拠」のリズムが明確になる。 |

## Part 2：ディスカッション結論（3スキル合意）

1. **ヒーロー**
   - 「おかげさまで実績 多数 件」～「ご依頼主様にご満足いただいています」のブロックは**廃止**。
   - **メインコピー**: 第三者目線の一声を1本立てる。  
     **案**: 「元警察官が代表の会社なので、安心してお任せしました。」（過去形で「依頼した人」の証言に）
   - サブ: 電話番号・受付時間・「無料お見積り」CTA、必要ならGoogle★5.0を短く1行で。

2. **ストーリーの流れ**
   - **FV**: 結論＝「安心して任せられる（権威：元警察官＋第三者証言）」→ 行動は「電話 or 見積り」。
   - **直下**: 共感（リード）で「家族で満足できる整理」を補強。
   - **その後**: 信頼の体制・Google口コミ・料金・流れ・FAQ・お客様の声 → 所々で「追加料金なし」「見積り後キャンセル無料」を繰り返し、最後にCTA。

3. **お客様の声**
   - 既存2件に加え、**「代表が元警察官だと知って安心して依頼した」** に近い内容の一声があれば1件追加（実際の口コミがあればそれを使用。なければ既存2件の並び順・見出しで「安心」を強調）。

4. **トーン**
   - 「ください」の押し売りより、「頂いています」「しました」のような**第三者・依頼済みの声**を前面に。企業の「私たちは」は控えめに。

## Part 3：変更内容の具体案（コピー・構成）

### 3-1. ヒーロー（差し替え案）

**削除する要素**
- 「おかげさまで実績」「多数 件」「奈良・関西エリアでご依頼多数」
- 「家族によりそう」「遺品・生前整理」（H1は残すが、メイン訴求からは外す or サブに回す）
- 「1R ご相談のうえお見積り 税込」
- 「Googleの口コミ ★5.0 ご依頼主様にご満足いただいています。」

**新構成案**

| 要素 | 内容 |
|------|------|
| **メインコピー（1本）** | 「元警察官が代表の会社なので、安心してお任せしました。」（第三者証言として） |
| **補足（1行）** | 例: 「奈良の遺品整理・生前整理」または「見積り無料・追加料金なし」 |
| **信頼バッジ** | Googleの口コミ ★5.0（短文のみ） |
| **CTA** | 電話番号 ＋ 「無料お見積り」ボタン（既存のフォーム or 電話への誘導） |

**H1**: 「遺品・生前整理」は**SEO・ページ主題**のため、サブ見出しまたは小さく残すことを推奨。

### 3-2. その他セクションの軽い手入れ（任意）

- **リード**: 冒頭に「自分たちでは手に負えない。でも、誰に頼めばいいかわからない。」のような**問題の一文**を入れる案。
- **認定「信頼の体制」**: 現状の「元大阪府警察官が代表を務めています。曲がったことが嫌いな性分です。見積り後の追加料金は、一切いただきません。」は**そのまま**でよい。
- **お客様の声**: 「代表が元警察官だと知り、安心して依頼しました」系の口コミがあれば1件追加。なければ見出しで「安心してお任せいただいています」を強調。

## Part 5：実装時の注意

1. **ヒーローの第三者証言**
   - 「安心してお任せしました」は**お客様の声として**表示するため、クォートや引用スタイル（「」やイタリック・名前の添え）があるとより自然。実際の口コミがあればそれに差し替え可能。

2. **H1とSEO**
   - 「遺品・生前整理」は`<h1>`として残し、メインコピーは`<p>`や`<blockquote>`で表現することを推奨。

3. **デザイン**
   - ヒーロー左側のテキスト量が減るため、余白とフォントサイズで**メインコピーを最も目立たせる**。

4. **計画書の提出**
